# タスク仕様: 表示要素共通化 + 名前表示モード

## 概要
以下の2点を実装する。
- アバター表示(`showAvatar`)と名前表示(`showAuthorName`)を全モード・全パネル状態で共通設定にする
- 名前表示を「@ハンドル」以外にも切り替えられるようにする

## 対象範囲
- 設定UI (`popup.html`)
- 設定正規化/保存 (`popup.js`, `config.js`)
- メッセージ解析 (`parser.js`)
- 表示描画 (`renderer.js`)
- 型コメント (`content.js`)
- 拡張バージョン更新 (PATCH)

## 追加仕様
- `authorNameMode` を追加する
  - `handle`: `@xxx` 形式で表示
  - `display-name`: 表示名を優先して表示（取得できない場合はハンドルへフォールバック）

## 共通化仕様
- `showAvatar` と `showAuthorName` は常に全プロファイル共通
- どのモードでトグルを変更しても全モードに即時反映
- 保存データ正規化時にも全プロファイルへ強制同期

## 受け入れ基準
- アバター表示/名前表示トグルを切り替えると、全モードの該当設定値が同一になる
- `authorNameMode` の選択が保存され、描画に反映される
- `display-name` 設定時、表示名が取得できれば `@` なしで表示される
- 不正値は `handle` にフォールバック
- 構文チェックとPlaywright検証でエラー0件

## 非対象
- 名前表示モードごとの色やフォントの個別設定
- YouTube側DOM仕様変更時の完全互換保証
