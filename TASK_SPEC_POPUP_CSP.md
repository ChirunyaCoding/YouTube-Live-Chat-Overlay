# タスク仕様: popup CSP違反の解消

## 目的
- 拡張機能 `popup.html` のインラインスクリプトを排除し、CSP違反エラーを解消する。

## スコープ
- `popup.html` からインライン `<script>` ブロックを削除する。
- 既存の `popup.js` に集約済みのタブ切替処理を利用する。
- `manifest.json` のバージョンをパッチ更新する。

## 制約
- 機能挙動（タブ切替・設定UI動作）は維持する。
- アイコン設定（`icons`, `action.default_icon` など）は追加しない。

## 受け入れ条件
- `popup.html` にインライン `<script>` が存在しない。
- ブラウザ検証でCSPのインライン実行エラーが発生しない。
- タブ切替が動作する。

## 対象外
- popup UIデザイン変更。
- 設定保存ロジックの仕様変更。
