# タスク仕様: 全画面設定導線と色設定の全サイズ共通化

## スコープ
- 全画面視聴中でも設定画面を開ける導線をオーバーレイUIに追加する。
- 名前色（メンバー/非メンバー）とコメント色を、全モード・全パネル状態で必ず共通値として扱う。
- 全画面導線で `ERR_BLOCKED_BY_CLIENT` が出る環境でも設定画面を開けるようにする。

## 制約
- 既存の拡張機能構成（`ui.js`, `popup.js`, `config.js`）を維持する。
- 既存のモード別/パネル別プロファイル構造は維持しつつ、色設定だけ強制共通化する。
- Chrome拡張マニフェストへアイコン設定は追加しない。

## 受け入れ条件
1. 全画面中に動画プレイヤー上から設定UI（`popup.html`）を開ける。
2. ポップアップで色設定を変更した時、モード/サイズ差異なく同じ色が適用される。
3. 旧設定データ（色がプロファイルごとに分かれている状態）を読み込んでも、正規化後は共通色に揃う。
4. 設定ボタン押下時に `chrome-extension://<id>/popup.html` 直接遷移によるブロックを回避できる。

## 非スコープ
- 色以外の既存項目の共通化ルール変更
- 設定画面の全面的なデザイン変更
