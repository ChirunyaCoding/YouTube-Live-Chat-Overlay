# タスク仕様: プレイヤー内専用設定ウィンドウ

## スコープ
- 動画プレイヤー内だけで表示される専用設定ウィンドウを追加する。
- 再生バーの「設定」ボタンでプレイヤー内ウィンドウを開閉できるようにする。
- 専用設定ウィンドウをプレイヤー内でドラッグ移動できるようにする。

## 制約
- 外部タブ遷移や `chrome-extension://...` 直接遷移に依存しない。
- 既存の設定UI（`popup.html`）を再利用する。
- 既存の編集ボタン（位置編集）機能は維持する。

## 受け入れ条件
1. 設定ボタン押下で、プレイヤー内に専用ウィンドウが表示される。
2. 専用ウィンドウはプレイヤー外では表示されない。
3. 専用ウィンドウを閉じる操作が提供される。
4. 既存のオーバーレイ描画・編集機能が壊れない。
5. 専用ウィンドウのヘッダーをドラッグして位置を変更できる。

## 非スコープ
- 設定フォーム項目自体の追加/削除
- 大規模なレイアウト刷新
