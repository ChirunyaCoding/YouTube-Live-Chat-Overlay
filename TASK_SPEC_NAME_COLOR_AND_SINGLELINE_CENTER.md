# タスク仕様: 名前色/コメント色設定と1行時の縦中央配置

## スコープ
- オーバーレイ表示で、1行メッセージ時にテキストブロックをアバター縦中央へ揃える。
- 設定画面で以下の色設定を追加し、保存・反映できるようにする。
  - 名前色（メンバー）
  - 名前色（非メンバー）
  - コメント色（共通）
- フェードイン/フェードアウトのアニメーションを自然な動きへ調整し、斜め上方向への不自然な遷移を解消する。
- 既存のモード/パネル状態ごとの設定保存方式を維持する。

## 制約
- 既存の拡張機能構成（`popup.html/js`, `config.js`, `renderer.js`, `parser.js`）を踏襲する。
- Chrome拡張のマニフェストにアイコン設定を追加しない。
- 設定未指定時は既存挙動に近い見た目を維持する。

## 受け入れ条件
1. 1行コメントでは、アバターとテキストが縦中央基準で揃う。
2. 複数行コメントでは、従来通り上寄せを維持する。
3. ポップアップから色を変更すると、保存後にオーバーレイへ反映される。
4. 名前色はメンバー/非メンバーで個別に設定できる。
5. コメント色は1つの設定で変更できる。
6. フェードイン/フェードアウト時に不自然な斜め上方向の遷移が発生しない。

## 非スコープ
- 新しい表示モードの追加
- 既存アニメーション仕様の大幅変更
- YouTube DOM依存ロジックの全面改修
